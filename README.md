# DIG DUG — 1982 Arcade Classic

A browser recreation of Namco’s **Dig Dug** (1982), companion to:

- [PAC-MAN](https://bamtec70.github.io/pacman-game/)
- [MS. PAC-MAN](https://bamtec70.github.io/ms-pacman-game/)

## Play

**Live:** https://bamtec70.github.io/dig-dug-game/

Or open `index.html` locally:

```powershell
start index.html
```

## How to play

Dig through the dirt, inflate enemies with your pump until they pop, and drop rocks on them for big points. Clear all monsters to advance.

### Desktop

| Control | Action |
|---------|--------|
| **Arrows / WASD** | Move & dig |
| **Space** (hold) | Pump / inflate enemy in front of you |
| **P** | Pause |
| **M** | Mute |

### Phone / touch

| Control | Action |
|---------|--------|
| **D-pad** | Move & dig |
| **P button** (hold) | Pump |
| **Swipe** on maze | Move |
| **❚❚ / ♪** | Pause / mute |

## Features

- Layered dirt & digging tunnels  
- **Pooka** (red) and **Fygar** (green, fire breath)  
- **Pump inflate → pop** scoring by depth  
- **Rocks** that fall and crush (bonus veggie)  
- Enemies enter **ghost mode** to pass through dirt  
- Levels with more enemies & rocks  
- Lives, high score, sound  
- Full **mobile / touch** controls  

## Files

- `index.html` — page shell  
- `style.css` — dirt-arcade framing + D-pad / pump button  
- `game.js` — full game engine
